Tutorial Singkat (WAJIB — jangan publish API key)
  1. Siapkan project di Vercel dan taruh file `public/index.html` ini serta dua serverless file di folder /api (lihat bawah).
  2. Tambahkan Environment Variable di Vercel: BOTCAHX_KEY = (API key Anda). JANGAN commit key ke git.
  3. Deploy project. Halaman akan otomatis panggil /api/checkkey dan /api/botchax (proxy).
  4. Untuk publik: API key selalu disimpan di env server; halaman **tidak** menampilkan atau mengirimkan apikey ke client.
